## windows terminal 快捷键

| 快捷键                           | 功能              | 备注 |
| -------------------------------- | ----------------- | ---- |
| shift + alt + "+"                | 横向分屏          |      |
| shift + alt + "-"                | 竖向分屏          |      |
| ctrl + shift + w                 | 关闭当前标签/分屏 |      |
| alt + 方向键                     | 分屏光标移动      |      |
| alt + shift + 方向键             | 调整当前分屏大小  |      |
| ctrl +shift + c                  | 复制              |      |
| ctrl + shift + v                 | 粘贴              |      |
| ctrl + shift + 1 - n             | 新建标签          |      |
| ctrl + alt + 1 - n               | 切换标签          |      |
| ctrl +鼠标滚轮                   | 调整窗口缩放      |      |
| ctrl + shift + pageup / pagedown | 翻页              |      |
| F11                              | 全屏              |      |
| ctrl + win + 上下键              | 窗口最大化切换    |      |
